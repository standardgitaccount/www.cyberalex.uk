Password Tester

The Password Entropy Estimator is a local calculator that estimates your password's entropy using a straightforward formula. It offers security recommendations and runs directly in the browser without transmitting any data.

Password Entropy Estimator

This calculator uses a simple formula to estimate the entropy of your password.
The minimum recommended entropy is 100 bits.

This will run locally on your machine and no data is saved or transmitted. An alternative is the Password Strength Testing Tool ↗ by Bitwarden.
