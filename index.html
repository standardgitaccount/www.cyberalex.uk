Integrated Document Viewer

View the test page for the seamless integration of a new document viewer for cyberalex.uk based on PDF.js, a Mozilla project and licensed under Apache 2.0. This project will enable easy viewing on desktops of PDF documents inside the webpage itself.

AWWC-DCA | January 2025

The Academic Weighted Word Count Calculator with Dynamic Count Allocation (AWWC-DCA) is a free-to-use tool that dynamically allocates a word count based on a given percentage that does not need to equal 100%. It is designed for university assignments.
> View Tool

Secure SME | Ongoing

SecureSME.co.uk is the end deliverable for the undergraduate cyber dissertation project. Designed to help inform small to medium enterprises in the UK of the requirements around data protection legislation and the importance of strong cyber security.
